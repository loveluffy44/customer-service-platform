# 客服管理平台原型

一个完整的客服管理平台前端原型，包含投诉处理、退款申请、订单查询、挽留审核等核心功能模块。

## 🚀 在线预览

访问 [GitHub Pages 链接](https://你的用户名.github.io/customer-service-platform/) 查看在线演示

## 📁 项目结构

```
.
├── index.html          # 主入口文件（客服管理平台原型）
└── README.md           # 项目说明
```

## 🛠️ 功能模块

### 1. 系统管理
- 账号管理
- 角色管理

### 2. 售后处理
- **投诉处理**：投诉列表、投诉详情、处理操作
- **退款申请表**：退款申请列表、全额/自定义退款
- **自动退款表**：自动退款记录查询
- **订单/用户查询**：支持订单号、用户ID、手机号查询
- **挽留审核列表**：挽留申请审核、附件管理

### 3. 数据报表
- 数据看板
- 渠道分析
- 绩效统计

### 4. 用户360视图
- 处理记录
- 用户信息
- 订单记录
- 产品行为记录
- 留言信息
- 400通话记录

## 📱 技术栈

- HTML5
- CSS3（原生样式，无框架依赖）
- JavaScript（原生 JS，无框架依赖）

## 🔧 本地运行

直接在浏览器中打开 `index.html` 文件即可预览。

或者使用本地服务器：

```bash
# Python 3
python -m http.server 8080

# Node.js
npx serve .
```

然后访问 http://localhost:8080

## 📄 使用说明

1. 点击左侧导航切换不同页面
2. 所有交互功能（弹窗、筛选、表格操作）均可正常使用
3. 数据为模拟数据，仅用于演示
